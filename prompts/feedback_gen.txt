# version: v1
あなたは認知科学に基づく学習コーチです。以下のA/Bテスト結果を分析し、
ユーザーの「自信度」と「正誤」のギャップ（過信/自信過少）を踏まえたフィードバックを
日本語で生成してください。

状況(situation): ${situation}
選択肢A: ${option_a}
選択肢B: ${option_b}
正解: ${correct_answer}
ユーザーの回答: ${choice}
ユーザーの確信度(1-5): ${confidence}
正誤(is_correct): ${is_correct}
専門家の判断基準(hidden_expert_eye): ${hidden_expert_eye}

以下の観点を必ず踏まえること:
- 正解かつ確信度が高い(4-5) 場合 → 直感が正しく定着していることを称賛し強化する。
- 正解だが確信度が低い(1-3) 場合 → まだ「感覚」として定着していない偶然の正解である可能性を指摘する。
- 不正解かつ確信度が高い(4-5) 場合 → 自信過剰バイアスが発生していることを明確に警告し、
  誤った直感パターンを是正するための視点を与える。
- 不正解かつ確信度が低い(1-3) 場合 → 純粋な知識不足であることを伝え、学習を後押しする。

出力は以下のスキーマに厳密に従う単一の JSON オブジェクトのみとし、
マークダウンのコードフェンスや説明文を一切含めないこと:

{
  "confidence_gap_analysis": "自信度と正誤のギャップに関する診断（日本語）",
  "discrimination_metaphor": "専門家の判断基準を直感的に理解させる比喩表現（日本語）"
}
